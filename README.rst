========
i2p.i2cp
========

public domain

requires:

* Python 2.6, 2.7 or 3.x
* I2P 0.9.14 or higher

to see it do stuff::

    git clone https://github.com/majestrate/python-i2cp
    cd python-i2cp
    pip install -r requirements.txt
    python -m i2p.i2cp

see examples directory for example usage
