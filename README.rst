========
i2p.i2cp
========

public domain

requires:

* Python 3.4 or higher
* I2P 0.9.14 or higher
* libsodium

to see it do stuff::

    git clone https://github.com/majestrate/python-i2cp
    cd python-i2cp
    pip install -r requirements.txt
    python -m i2p.i2cp

see examples directory for example usage
